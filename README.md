# BeatForge AI

BeatForge AI is a procedural music generation tool that runs entirely in the browser. It allows users to create professional-quality beats and musical tracks using a 12-track sequencer and AI-powered generation algorithms.

## Features

*   **AI Generation**: Instantly generate beats based on genres like Trap, Lo-Fi, Techno, Synthwave, and more.
*   **12-Track Sequencer**: Full control over your composition with a 16-step sequencer.
*   **Procedural Audio**: All sounds are synthesized in real-time using the Web Audio API. No samples required.
*   **Professional Effects**: Built-in Reverb, Delay, Chorus, and EQ.
*   **Export to WAV**: Download your creations as high-quality WAV files.
*   **Responsive Design**: Works on desktop and mobile devices.

## Technologies Used

*   **HTML5 & CSS3**: Semantic markup and modern styling.
*   **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
*   **JavaScript (ES6+)**: Core logic and interaction.
*   **Web Audio API**: Real-time audio synthesis and processing.

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/beatforge-ai.git
    ```
2.  Open `index.html` in your web browser.
3.  Start creating music!

## Usage

1.  Navigate to the **Beat Maker** section.
2.  Select a genre and click **Generate AI** for instant inspiration.
3.  Manually adjust the grid to refine your beat.
4.  Use the **Track Settings** to customize instruments and effects.
5.  Click **Export** to save your track.
